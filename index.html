Microsoft

Software Engineer Intern, 2019-2020

During my time at Microsoft, I worked as a Software Engineer Intern on the Azure Cloud Hardware and Infrastructure Engineering (CHIE) Firmware team. My team was responsible for developing and deploying the Windows Cloud Server system firmware (i.e. the low-level software that initializes and controls the server hardware during bootup and runtime).

SELinator

One key feature in our firmware implementation is the system event log (SEL), which maintains a record of all hardware-related events and errors. This telemetry comes in handy when a server goes down and our engineers need to diagnose the root cause of the failure. The SEL is stored as a stream of raw bytes, and prior to my arrival at Microsoft, CHIE engineers would have to manually decode the SEL by referencing the appropriate platform specification, which was a tedious process.

My project for the summer of 2019 was to implement a platform-agnostic SEL parsing tool (creatively titled "SELinator") that automated the decoding and filtering of raw SEL data in order to help streamline the redeployment of out-for-repair servers. Once I completed the bare-bones offline console application, I took the project to the next level of accessibility by hosting SELinator in Azure, designing an interactive bot user interface for the app, and integrating it into Microsoft Teams.

I can tell that SELinator has truly made an impact because when I came back to Microsoft a year later in 2020, employees across the entire Azure enterprise were using the tool for their day-to-day tasks.
